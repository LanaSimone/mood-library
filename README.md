# Mood Library

Mood Library is a full-stack web application that allows users to organize songs into mood-based playlists like "happy", "chill", or "late night".

## Tech Stack

Frontend:
- React (Vite)
- React Router
- JavaScript
- CSS

Backend:
- Node.js
- Express
- SQLite

## Features

- Add songs with title, artist, and mood
- View all songs in a library
- Mood-based organization (playlist-style)

## In Progress

- Connecting frontend to backend API
- Fetching moods dynamically from database
- User authentication (login/signup)
- Persisting songs per user
- Mood-based playlist views
